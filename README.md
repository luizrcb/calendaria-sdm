## Instalation

You can use one of the following installation methods:

1. Pasting the following url into the **Install Module** dialog on the Setup menu of the application.
##
    https://github.com/luizrcb/calendaria-sdm/releases/latest/download/module.json
2. Browsing the repository's [Releases](https://github.com/luizrcb/calendaria-sdm/releases) page, where you can copy any module.json link for use in the Install Module dialog.
3. Downloading one of the .zip archives from the Releases page and extracting it into your foundry Data folder, under `Data/modules/calendaria-sdm`.
